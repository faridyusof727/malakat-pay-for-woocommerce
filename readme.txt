== Description ==

Plugin to enable payment for your woocommerce through raudhah pay.

== Installation ==

**Step 1:**

Register Raudhahpay account at here https://cloud.raudhahpay.com/user/register

**Step 2:**

Login to your WordPress Dashboard
Navigate to Plugins >> Add New
Search RaudhahPay for WooCommerce >> Install Now

**Step 3:**

Activate Plugin

**Step 4:**

Navigate to WooCommerce >> Settings >> Payments >> Raudhahpay
Insert your API Access Token, API Signature Key and Collection ID then click Save changes

== Frequently Asked Questions ==


= Where can I get the API Key and Signature Key? =

Yes you can, just sign in to raudhahpay then navigate to Organization Setup > Api Settings

= Do Raudhpay have sandbox environment? =

Yes we do have, please go to https://stg.raudhahpay.com and register there.
